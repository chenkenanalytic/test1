# 你比想像中還強大

有時候，生活像是一場不知所措的長跑，喘不過氣、看不到終點，覺得自己快要撐不下去。但你知道嗎？你已經走了很遠很遠，這一路上每一個你以為撐不過去的瞬間，都在悄悄地讓你變得更堅強。

你懷疑過自己，曾經迷路，也曾想放棄。可你還是咬牙站了起來，繼續往前。你比你以為的還勇敢，或許你現在感覺不到，但那些困難、那些傷心和彷徨，其實都是你成長的證明。

不要被短暫的低潮擊倒，也別讓別人的評價左右你的內心。這世界很大，未來更充滿可能，可是只有你能定義你的價值。今天只需要比昨天更真實地相信自己一點點，日子就會悄悄轉晴。願你在人生的馬拉松裡，再走一步，再喘一口氣，發現原來自己真的很強大。你不是一個人，這條路上比你想像中還多溫柔和希望。
